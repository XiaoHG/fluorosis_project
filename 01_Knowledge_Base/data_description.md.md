# 可用数据集描述

## 1. 氟斑牙（Dental Fluorosis）图像数据集

### 1.1 数据概览
- **图像总数**：200 张
- **分级体系**：Dean 指数简化版（0–3 级）
  - 0 – Normal（正常）
  - 1 – Mild（轻度）
  - 2 – Moderate（中度）
  - 3 – Severe（重度）
- **每类样本数**：各 50 张，类别完全平衡。

### 1.2 图像属性
- **模态**：口腔自然光照片 
- **图像格式**：png
- **分辨率/尺寸**：512x256

### 1.3 标注信息
- **标注来源**：由临床医生根据 Dean 氟斑牙指数分级标准进行人工标注。

### 1.4 文件组织结构
dental_fluorosis/images/
├── normal/ (50 images)  
├── mild/ (50 images)  
├── moderate/ (50 images)  
└── severe/ (50 images)

## 2. 氟骨症（Skeletal Fluorosis）X 光图像数据集

### 2.1 数据概览
- **X 光图像**：80 张
- **对应 Mask（分割标签）**：59 张（部分图像有 mask，另外一部分是无患病的正常样本）
- **任务类型**：
  - 二分类：是否患有氟骨症（0/1）
  - 四分类（严重程度分级）：0 – 正常，1 – 轻度，2 – 中度，3 – 重度
- **额外标注**：
  - 每位患者同时有 5 位放射科医生的人工诊断结果
  - 数据划分标签（训练/测试）已提供

### 2.2 图像属性
- **模态**：X 光平片，口腔图像
- **图像格式/分辨率/尺寸****：口腔图像512x256的png图像，氟骨症X光512x1024的png图像
- **是否已脱敏**：是

### 2.3 Mask 数据
- **Mask 类型**：骨骼区域分割 / 病灶区域标注 
- **对应关系**：59 张 mask （其他没有mask的图像是无患病的正常图像）与 80 张 X 光图像的对应方式：按文件名对应

### 2.4 标注文件说明

#### 2.4.1 GT.xlsx
综合标注文件，包含以下字段：

| 字段名     | 说明                            |
| ------- | ----------------------------- |
| `患者ID`  | 患者唯一标识符                       |
| `二分类`   | 0: 正常，1: 患有氟骨症                |
| `多分类`   | 严重程度分级：0=正常, 1=轻度, 2=中度, 3=重度 |
| `训练/测试` | 指定该样本用于训练集还是测试集               |
| `身体部位`  | 拍摄部位（如前臂、小腿、骨盆等）              |

#### 2.4.2 放射科医生独立诊断结果
共 5 位医生，每位医生一个 Excel 文件（命名规则 `医生姓名.xlsx`），每个文件包含：

| 字段名 | 说明 |
|--------|------|
| `图片编号` | 对应 X 光图像文件名或 ID |
| `是否患有氟骨症` | 0: 否，1: 是 |
| `严重程度` | 0: 正常, 1: 轻度, 2: 中度, 3: 重度 |

**用途**：
- 可用于计算标注一致性（如 Fleiss' Kappa），确定真实标签时可能采用多数投票或专家共识。
- 可评估模型与人类医生表现的对比。

## 3. 数据挑战与注意事项

1. **氟斑牙数据集样本量较小**（每类 50 张，共 200 张），需考虑数据增强、迁移学习等手段防止过拟合。
2. **多位医生标注**提供了标签不确定性信息，设计实验时可考虑将医生间一致性作为参考上限。
3. **训练/测试划分已在 GT.xlsx 中指定**，实验时应严格遵循，避免信息泄露。
4. **类别分布**：氟斑牙已人为平衡；氟骨症类别分布待统计（可从 GT.xlsx 分析得出）。
