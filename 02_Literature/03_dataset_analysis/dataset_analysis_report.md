## 五、对 EDL + ORCU 项目的启示

### 5.1 机遇

1. **标注不确定性是核心卖点**: 五位医生四分类平均一致率仅 **54.1%**，最强证据支持 EDL 不确定性建模的必要性。Mwinc-Mamba 论文虽报告了 poor inter-rater agreement，但未进行数值化统计 — 我们的分析结果可以作为 Introduction 的核心论据。

2. **SOTA baseline 明确**: Mwinc-Mamba (SF: 66.67% 四分类) 和 MLTrMR (DF: 80.19% Acc) 是需要超越的目标。EDL + ORCU 提供了差异化创新点。

3. **多部位 X 光是挑战也是机会**: SF 数据含前臂/小腿/骨盆三种部位，姿态和解剖结构差异大。EDL 的 evidence 分布可为不同部位提供自适应不确定性估计。

4. **数据集可直接用于实验**: 无需额外数据采集或标注 — 直接可用。

### 5.2 挑战

1. **样本量小**: SF 80 张 (训练仅 56) — EDL 在小样本下的不确定性估计需仔细验证；DF 200 张相对充裕。

2. **类别不平衡**: SF 的 Mild 占 42.5%，Severe 仅 15% — ORCU 的有序回归约束在此分布下需调整权重。

3. **图像预处理差异**: 原始 DICOM → 8-bit PNG 已丢失信息 — 可能需要请求原始 DICOM 数据以获得最佳性能。

4. **ZY 医生的异常标注**: ZY 与 GT 的 4-grade 一致率仅 48.8% — 是否应排除这位医生的标注？或在 EDL 中建模为额外的噪声源？

## 六、结论

该数据集质量高、文档齐全、与我们的项目目标完全对齐。关键数据点:

- **DF**: 200 张四分类, 平衡, 可直接训练
- **SF**: 80 张三部位四分类, 含 5 位医生标注 (可计算 EDL evidence)
- **最大创新支撑**: 54.1% 的放射科医生四分类平均一致率直接验证了 EDL 的必要性
- **SOTA target**: DF 80.19% / SF 66.67% — EDL + ORCU 有望超越
