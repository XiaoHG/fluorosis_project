# 遵义市新民镇朝阳村 2009 与 2023 年地氟病调查分析

## Metadata

| Field | Value |
|-------|-------|
| Authors | 顾茂华, 吴云, 王世杰, 江中川 |
| Venue | 中国地方病防治 (Chin J Ctrl Endem Dis), Vol. 39, No. 3, pp. 189-191 |
| Year | 2024 |
| Affiliation | 贵州大学 + 遵义市播州区南白社区卫生服务中心 |
| Funding | 国家自然科学基金 (62266011) |

## Summary

燃煤污染型地氟病流行病学调查。以贵州省遵义市新民镇朝阳村为调查点，对比 2009 与 2023 年防治成效。调查包括 73 名 8-12 岁小学生氟斑牙筛查、居民问卷调查、氟骨症患者随访。

### Key Results
| 指标 | 2009 | 2023 | 变化 |
|------|------|------|------|
| 8-12岁学生地氟病知晓率 | 100% | 23.3% | ↓ 76.7% |
| 8-12岁学生氟斑牙检出率 | 33.7% | 16.4% | ↓ 17.3% |
| 居民正确使用炉灶率 | 0% | 61.6% | ↑ 61.6% |
| 正确干燥辣椒率 | 33.3% | 78.1% | ↑ 44.8% |
| 正确干燥玉米率 | 30% | 78.1% | ↑ 48.1% |

### Conclusion
- 地氟病防治已达控制水平，儿童氟斑牙患病率从 33.7% 降至 16.4%
- 但知晓率反而下降（100% → 23.3%），居民健康行为未完全转变
- 需持续加强宣教以巩固防治成果，最终实现完全消除

- 证实贵州大学吴云团队在氟中毒领域完整布局：流行病学 → DF/SF 数据集 → DL 诊断模型
