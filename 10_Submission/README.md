## Submission Checklist (Elsevier MedIA)

- [ ] Manuscript (PDF, LaTeX source)
- [ ] Cover Letter
- [ ] Highlights (3-5 bullets, max 85 chars)
- [ ] Graphical Abstract (image)
- [ ] Supplementary Material (PDF)
- [ ] Author Declaration / Conflict of Interest
- [ ] Data Availability Statement
- [ ] Code Availability (GitHub link)
- [ ] Suggested Reviewers (3-5)
- [ ] Opposed Reviewers (if any)

## Pre-Submission Final Checks

1. [ ] All [X] placeholders filled with experimental data
2. [ ] All references complete (no [Author]/[Journal]/[Year])
3. [ ] All figures inserted and cross-referenced
4. [ ] Author list finalized and approved by all co-authors
5. [ ] Funding and ethics statements completed
6. [ ] Code repository cleaned and README updated
7. [ ] Supplementary PDF compiled
8. [ ] Manuscript within page/length limits
9. [ ] Plagiarism check passed
10. [ ] Final proofread by all authors
