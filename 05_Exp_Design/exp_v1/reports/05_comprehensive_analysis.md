# DF Fluorosis 跨版本综合实验分析

## 版本概览

| 版本 | 日期 | 环境 | 状态 | 用途 |
|------|------|------|------|------|
| v2.0 | 05-16 | Kaggle (PyTorch 2.5) | 原始基线 | 参考 (有架构 bug: CE/SORD 错误使用 EDLClassifier) |
| v2.1 | 05-16 | Kaggle (PyTorch 2.5) | Bug 版本 | 参考 (EDL %Unim 崩塌至 28.3%) |
| **v2.2** | **05-16** | **Kaggle (PyTorch 2.5)** | **✅ 论文定稿** | **主实验 + Multi-seed + 5-Fold CV** |
| v5 | 05-17 | Kaggle (PyTorch 2.6) | 退化 | 废弃 (EDL 58.33%, 环境变化) |
| v6 | 05-17 | Kaggle (PyTorch 2.6) | Per-sample 数据 | 图表生成 (聚合指标不可用于论文) |

## 核心结论

**v2.2 为论文唯一定稿数据源。** v2.2 拥有正确的模型架构 (StandardClassifier for CE/SORD)、最高的 EDL 精度 (83.33%)、最低的 ECE (0.0719)、以及完善的 Multi-seed + 5-Fold CV 验证。

## 九、论文叙事线

### 强结论

1. **CE baseline (81.67%) 已超越全部竞品** — ImageNet-21k 预训练 ViT 是关键
2. **EDL (83.33%) 新 SOTA** — 比 MLTrMR 高 +3.1pp Acc, +0.125 QWK
3. **首次在氟斑牙诊断中提供校准不确定性** — ECE=0.072, 竞品均无校准
4. **Multi-seed: EDL CV=4.3% vs CE CV=31.4%** — KL 正则化赋予初始化鲁棒性
5. **EDL+ORCU CV QWK std=0.32%** — 跨三版本最稳定
6. **SORD 100% 单峰** — 临床可解释性安全选择

### 需诚实讨论的局限

1. **EDL AUROC(u) < 0.5 三版本一致** — 不确定性不能用于临床拒绝
2. **仅 DFID 单数据集** — 外部验证待做
3. **Cumulative 跨版本波动** — QWK 0.36-0.92, 欠鲁棒
4. **Lambda/Temp sweep 在 v6 环境** — 绝对值偏保守

### 潜在审稿争议预判

| 争议 | 回应 |
|------|------|
| ViT CE 81.7% vs LD2Net ViT 44.1% | ImageNet-21k 预训练; 明确报告环境和配置 |
| EDL 83.3% vs v2.0 85.0% | v2.0 架构有 bug, v2.2 是正确实现 |
| 为什么 PyTorch 2.6 下退化 | 注明 v2.2 使用 PyTorch 2.5, 给出 seed 配置 |
| 3 seed 是否够 | CV=31.4% 已充分暴露 CE 脆弱性 |
| AUROC(u) < 0.5, 不确定性有用吗 | 区分校准 (ECE ✓) 和错误检测 (AUROC ✗) |
